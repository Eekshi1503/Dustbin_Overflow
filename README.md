# Smart Dustbin Overflow Detection System

## Overview
This project is a smart waste management system that detects dustbin overflow using Computer Vision techniques. It helps in maintaining cleanliness by identifying when a dustbin is full and needs to be emptied.

## Features
- Real-time dustbin monitoring using camera input
- Detects garbage level using object detection
- Alerts when dustbin is full
- Helps in smart city waste management
- Reduces manual inspection effort

## Tech Stack
- Python
- OpenCV
- YOLOv8
- NumPy

## How It Works
1. Capture image/video from camera
2. Detect garbage/dustbin using YOLO model
3. Analyze fill level
4. Trigger alert if overflow is detected
